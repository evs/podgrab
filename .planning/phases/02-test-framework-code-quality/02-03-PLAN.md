</acceptance_criteria>

<action>
In `main.go`, replace lines 44-50 (the entire `"removeStartingSlash"` entry in funcMap) with:
```go
"removeStartingSlash": func(raw string) string {
    if len(raw) > 0 && raw[0] == '/' {
        return raw[1:]
    }
    return raw
},
```

This:
1. Removes the `fmt.Println(raw)` debug print (BUG-03 fix)
2. Reverses the logic: when a leading "/" is present, return `raw[1:]` (strip it); otherwise return `raw` unchanged (BUG-02 fix)
3. Adds `len(raw) > 0` guard to prevent index-out-of-range panic on empty string input (the old `string(raw[0])` pattern would panic on `""`)
</action>
